== Description ==
The plugin add a navigator to posts so there are anchors to each H2 title from the top of the post.

== Installation ==

1. Upload `rotem-navigator.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the shortcode [rotemnavigator] in the post you want to show the navigator.
